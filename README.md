# OIBSIP_Basic-Network-Scanning-with-Nmap
Performed a basic network scan on the local host (127.0.0.1) using Nmap to identify open ports and running services and analyzed the security significance of the exposed services.

# Environment
- OS: Kali Linux (Virtual Machine)
- Target: Localhost (127.0.0.1)
- Authorization: Self owned system

# Tools Used
- Nmap (Network Mapper)

# Methodology
1. Verified Nmap installation and confirmed the installed version.
2. Confirmed localhost reachability to ensure the target was accessible.
3. Performed service and version detection on the target host using Nmap.
4. Saved and documented the scan output for analysis and record-keeping.
